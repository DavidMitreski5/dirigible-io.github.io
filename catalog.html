Applications Catalog

Applications Catalog is the place where the Dirigible community can share, learn and collaborate on the projects built-with and run-on Eclipse Dirigible. There are several groups of applications depending on their type (core, sample, template, extension) and as well as on their industry related context. Applications Catalog is supposed to be not only a simple list, where the projects can be linked for download or import, but also a place where the project owners can advertise their work and skills in Eclipse Dirigible and in the various business areas.

To link your project under the Applications Catalog, you can make a pull request with a page describing your project with the mandatory information like:

  • author - the GitHub user or GitHub organization
  • repository - a Git based source code repository
  • license - preferred are the more permissive licenses e.g Apache v2, MPL, MIT. If you plan your project to become part of the core of Eclipse Dirigible, it is good to use EPL from the beginning.
For reference you can use core_api

Only projects with a valid and well known open-source license will be approved. GPL, LGPL and AGPL will be rejected as of now due to legal issues, which prevents such projects to be inclued in commercial products with closed source in the future.
