Legacy Components

Variety of integration patterns for your legacy components are possible. Java libraries and their APIs can be made available from within the scripting services once they are bundled along with the standard Dirigible's runtime package (WAR) using the build framework of your choice e.g. Maven. non-Java libraries and services can be connected either as external processes or as 3-thd party services in integration routes.

Adaptable Software

Based on the In-System Programming Model and accessible worldwide directly via a Browser only, the developers using Dirigible have the freedom and power to follow business requirements just right away. The applications built with the cloud toolkit complies with the Dynamic Applications paradigm, therefore they are built for change.
