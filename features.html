Fast Prototyping

Shortest turn-around time, as well as generation of predefined ready-to-use templates at any level of application development, empowers you to build a skeleton for your project directly in front of your eyes.

More Info

Legacy Components

Variety of integration patterns for your legacy components are possible. Java libraries and their APIs can be made available from within the scripting services once they are bundled along with the standard Dirigible's runtime package (WAR) by using the build framework of your choice, e.g. Maven. Non-Java libraries and services can be connected either as external processes or as third party services in integration routes.

More Info

Adaptable Software

Given that Dirigible is based on the In-System Programming Model and is directly accessible world-wide via only a browser, developers have the freedom and power to follow business requirements just right away. The applications built with the cloud toolkit complies with the Dynamic Applications paradigm, therefore they are built to be changed.

Database Development

  • Dirigible enables you to model data structure artifacts in JSON format, and to create actual tables and views during activation/publishing.
  • Database perspective is full-fledged including Database Explorer view, SQL Console, meta-data inspectors, and more.
  • You can autogenerate mock-up data based on given table layout.
More Info

Server-Side Development

  • Dirigible uses some of the most popular dynamic languages (such as JavaScript, Ruby, and Groovy) for creating server-side services.
  • It supports the standard CommonJS modularization.
  • It generates fully compliant RESTful services, including meta-data support on existing database artifacts.
More Info

Client-Side Development

  • Dirigible supports user interface generation for the RESTful services based on widely used frameworks, such as Bootstrap, jQuery, AngularJS, and OpenUI5.
  • It provides WYSIWYG editor for both standard HTML5 and Bootstrap-based complex controls.
  • It supports wiki scripting via Confluence format.

Connecting to External Services

Dirigible provides built-in micro ESB based on Apache Camel and CXF for declarative integration and orchestration of the business process flows.
