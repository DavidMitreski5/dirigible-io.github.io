Operational Service
===

Operational Service exposes some utility functions.

> The end-point is: */op*

* To get the current logged-in user name:


> Parameter: *user*
> 
> http //[host]:[port]/dirigible/ *op?user*

* To log-out from the the current user session:

> Parameter: *logout*
> 
> http //[host]:[port]/dirigible/ *op?logout*
