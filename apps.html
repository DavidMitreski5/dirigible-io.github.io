Samples

Sample applications show specific aspects of the usage of Eclipse Dirigible and Enterprise JavaScript API. There are two types of samples in this category:

  • Simple Samples - demoing a single feature or a single API usage. They are supposed to be highly reusable code snipets.
  • Solution Samples - showing more complex use-cases much closer to the real world applications.

Extensions

Extension applications are pluggins, which enhance the functionality of the Eclipse Dirigible itself. They are not supposed to be an integral part of Dirigible, but can be easily imported and used by the end-users on their own instances. Applications in this category are developed by the Dirigible Labs teams, but have not passed (nor their dependencies) the Eclipse IP Process.

Applications Hub

Applications Hub is a place where the Dirigible community can share, learn and collaborate on the projects built-with and run-on Eclipse Dirigible. There are several groups of applications depending on their type (core, sample, template, extension) and as well as on thier origin and license. Applications Hub on dirigible.io site is supposed to be not only a simple catalog, where the projects can be linked for download or import, but also a place where the project owners can advertise their work and skills in Eclipse Dirigible and the various business areas.

To link your project under the Applications Hub, you can make a pull request with a page describing your project with the mandatory information like:

  • author - the GitHub user or GitHub organization
  • repository - a Git based source code repository
  • license - preferred are the more permissive licenses e.g Apache v2, MPL, MIT. If you plan your project to become part of the core of Eclipse Dirigible, it is good to use EPL from the beginning.
For reference you can use core_api

Only projects with a valid and well known open-source license will be approved. GPL, LGPL and AGPL will be rejected as of now due to legal issues, which prevents such projects to be inclued in commercial products with closed source in the future.
